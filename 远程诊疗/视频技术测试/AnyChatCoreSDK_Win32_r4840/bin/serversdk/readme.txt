该目录下的可执行程序均为业务服务器程序（调用AnyChat Server SDK），需与核心服务器配合才能正常工作。

1、AnyChatBussinessServer(C++)	调用AnyChat Server SDK的演示程序，源代码在src\server\c++目录中

2、AnyChatBussinessServer(C#)	调用AnyChat Server SDK的演示程序，源代码在src\server\c#目录中

3、AnyChatBusinessServer(Java)	调用AnyChat Server SDK的演示程序，有GUI界面，源代码在src\server\java目录中

4、AnyChatSampleServer(Java)	调用AnyChat Server SDK的演示程序，没有GUI界面，适合在控制台运行（RunAnyChatSampleServer.bat），源代码在src\server\java目录中

5、AnyChatCallCenterServer(C#)	视频呼叫中心业务服务器，实现了视频呼叫和大厅好友等功能，源代码在src\client\c#目录中

6、AnyChatCallCenterServer(Java) 视频呼叫中心业务服务器，实现了视频呼叫和大厅好友等功能，没有GUI界面，适合在控制台运行（RunAnyChatCallCenterServer.bat），源代码在src\server\java目录中

7、AnyChatCallCenterServer(C++) 视频呼叫中心业务服务器，实现了视频呼叫和大厅好友等功能，源代码在src\server\c++目录中

8、x64\				Windows 64bit运行库，适用于JDK 64bit（JRE 64bit）等环境下使用
