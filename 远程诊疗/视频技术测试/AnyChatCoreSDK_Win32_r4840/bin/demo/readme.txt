1、AnyChat Platform Core SDK 在正式部署时，需要将组件[bin\client目录下的所有文件]拷贝到.exe的当前目录，或是video子目录下即可，参考FAQ：http://www.anychat.cn/faq/index.php?action=artikel&cat=7&id=242&artlang=zh

2、BRAnyChatSDKDemo.exe VC++演示程序，演示了客户端SDK的应用，可以实现多方语音、视频的交互；
	（源代码所在目录：demo\client\C++，编译环境：VC++6.0、VS2008）

3、AnyChatDemo(VB.Net).exe VB.Net演示程序，演示了客户端SDK在VB.Net平台上的应用，可以实现点对点的语音、视频交互以及文字交流；
	（源代码所在目录：demo\client\VB.Net，编译环境：VS2008）

4、AnyChatDemo(C#).exe	C#演示程序，演示了客户端SDK在C#平台上的应用，可以实现点对点的语音、视频交互以及文字交流；
	（源代码所在目录：demo\client\C#，编译环境：VS2008）

5、BRAnyChatMeeting.exe VC++演示程序，一个简单的视频会议软件，可用于评估AnyChat的语音、视频质量
	（没有开放源代码）

6、AnyChatDemoForJava	Java演示程序，演示了客户端SDK在Java平台上的应用，源代码在SDK包的：src\client\java目录下

7、AnyChatCallCenterClient(C#)	视频呼叫中心客户端，实现了视频呼叫和大厅好友等功能，源代码在src\client\c#目录中
